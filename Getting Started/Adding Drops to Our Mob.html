Drop Tables

We are now going to configure the drops for our mob. MythicMobs allows us to configure the mob's drops in a very flexible and powerful way.

We will start by adding a Drops section to our mob's configuration file.

/plugins/MythicMobs/Mobs/DemonicMage.yml
DemonicMage:
  Type: SKELETON
  Display: "&4☠ &7Demonic Mage &4☠"
  Health: 100
  Damage: 10
  Options:
    MovementSpeed: 0.3
    PreventOtherDrops: true
    PreventSunburn: true
    KnockbackResistance: 1
  Equipment:
  - BLAZE_ROD HAND
  - LEATHER_HELMET HEAD
  - LEATHER_CHESTPLATE CHEST
  - LEATHER_LEGGINGS LEGS
  - LEATHER_BOOTS FEET
  Skills:
  - effect:particles{p=flame;hS=0.25;vS=0.5;amount=10} @selfLocation{yoffset=1} ~onTimer:5
  - skill{skill=DemonicMageFireball} ~onTimer:20 0.5
  Drops:
  - ...

We will add our drops to this section in the form of a YAML list. Each entry in the list will represent a single drop.

Drops:
- blaze_powder 3-5 0.5

Let's break this down into its components.

  • blaze_powder is the item that will be dropped.
  • 3-5 is the amount of items that will be dropped. This will be a random number between 3 and 5.
  • 0.5 is the chance that this drop will occur. This will be a 50% chance.

We can add as many drops as we want to this list.

Drop Tables​

The above drop format is very simple, but its capabilities are limited. To make full use of MythicMobs' drop system, we will need to use drop tables.

Let's go back to our MythicMobs folder, which should now look similar to the following structure below. We will now visit a new folder called DropTables:

MythicMobs folder

We will create a new file called DemonicMageDrops.yml in this folder.

The format of a drop table is very similar to the format of a skill:

/plugins/MythicMobs/DropTables/DemonicMageDrops.yml
DemonicMageDrops:
  Drops:
  - blaze_powder 3-5 0.5
  - fire_charge 1-2 0.2

Let's break this down into its components. Like a skill config or a mob config, the drop table has an ID at the top. The actual drops are defined in the Drops section.

This allows for us to group our drops into different drop tables, and then reference them in our mob configs. In this case, let's add it to our mob configuration. To do so, add a new drop like you would normally, except instead of writing the item, simply write the ID of the drop table.

/plugins/MythicMobs/Mobs/DemonicMage.yml
DemonicMage:
  Type: SKELETON
  Display: "&4☠ &7Demonic Mage &4☠"
  Health: 100
  Damage: 10
  Options:
    MovementSpeed: 0.3
    PreventOtherDrops: true
    PreventSunburn: true
    KnockbackResistance: 1
  Equipment:
  - BLAZE_ROD HAND
  - LEATHER_HELMET HEAD
  - LEATHER_CHESTPLATE CHEST
  - LEATHER_LEGGINGS LEGS
  - LEATHER_BOOTS FEET
  Skills:
  - effect:particles{p=flame;hS=0.25;vS=0.5;amount=10} @selfLocation{yoffset=1} ~onTimer:5
  - skill{skill=DemonicMageFireball} ~onTimer:20 0.5
  Drops:
  - DemonicMageDrops

Nice! Now our mob will drop the items defined in the drop table.

Additional capabilities​

Besides grouping drops, drop tables have a few other capabilities that we will look into. For example, we want to make our mob only drop either a blaze powder or a fire charge, but never both. We can do that by adding the TotalItems option to our drop table.

/plugins/MythicMobs/DropTables/DemonicMageDrops.yml
DemonicMageDrops:
  TotalItems: 1
  Drops:
  - blaze_powder 3-5 0.5
  - fire_charge 1-2 0.2

In addition, we want to make our mob's drops scale on the player's luck. The BonusLuckItems option allows us to add a random amount of items to the drop table, based on the player's luck.

/plugins/MythicMobs/DropTables/DemonicMageDrops.yml
DemonicMageDrops:
  TotalItems: 1
  BonusLuckItems: 1
  Drops:
  - blaze_powder 3-5 0.5
  - fire_charge 1-2 0.2

In this case, if the player has, for example, 3 luck, the drop table will drop 6-8 blaze powder or 4-5 fire charges.

Conclusion​

Not only have we removed the default drops from our mob, but we have also added a drop table that allows us to drop multiple items at once, and scale it based on the player's luck. Having a drop table also allows us to easily reuse the drops in other mobs, and makes it easier to maintain our drops.
